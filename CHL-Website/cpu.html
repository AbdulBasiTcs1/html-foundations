What is the CPU?

The CPU (Central Processing Unit) is the most important component of a computer. It is often called the "brain of the computer" because it performs all calculations and executes all instructions given by software. Every program you run ‚Äî from opening a file to playing a game ‚Äî is processed by the CPU. Modern CPUs are very small chips but contain billions of tiny transistors.

The CPU follows a simple cycle called the Fetch-Decode-Execute cycle:

  • Fetch ‚Äî The CPU fetches the next instruction from memory (RAM).
  • Decode ‚Äî The CPU decodes the instruction to understand what to do.
  • Execute ‚Äî The CPU carries out the instruction (e.g., add two numbers).
"The CPU is the brain of the computer ‚Äî every single operation the computer performs passes through it."

What is the ALU (Arithmetic Logic Unit)?

The ALU is a part of the CPU that performs two types of operations:

  • Arithmetic Operations: Addition, Subtraction, Multiplication, Division
  • Logical Operations: AND, OR, NOT, XOR (used for comparisons and decisions)

Every time you add numbers, compare values, or make a decision in a program, the ALU is doing that work. The result of ALU operations is either stored back in a register or sent to memory.
