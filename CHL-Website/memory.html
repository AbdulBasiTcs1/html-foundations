Detailed Descriptions

RAM (Random Access Memory)

RAM is the main memory used by the computer while it is running. When you open an application like a web browser or a game, it is loaded from storage into RAM so the CPU can access it quickly. More RAM means you can run more programs at the same time without slowing down. Modern computers use DDR4 or DDR5 RAM.

ROM (Read Only Memory)

ROM stores the BIOS (Basic Input Output System) â€” a small program that starts the computer when you press the power button. It checks all hardware, then loads the operating system. Unlike RAM, ROM does not lose its data when the computer is turned off.

Cache Memory

Cache is ultra-fast memory located inside or very close to the CPU. It stores copies of recently used data so the CPU does not have to go all the way to RAM to find it. Cache is organized in levels: L1 (smallest, fastest), L2, and L3 (largest, slightly slower).
