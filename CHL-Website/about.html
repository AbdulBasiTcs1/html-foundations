About This Website

Computer Hardware Learning (CHL) is an educational website created as part of the Web Technology (Lab Assignment 1) course at COMSATS University Islamabad, Attock Campus, Department of Computer Science, Spring Semester 2026.

The purpose of this website is to provide beginner-level, easy-to-understand information about the different components of computer hardware. Topics include Input Devices, Output Devices, the CPU, ALU, Memory (RAM & ROM), Storage Devices, and the Motherboard.

This website is designed using basic HTML and inline CSS, keeping the code simple and easy to understand for students who are just beginning their journey in web development.
