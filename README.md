# html-foundations
Complete HTML5 foundations with practical examples, best practices, and mini projects for building real-world web pages.

## Usage

- Open html_practice.html in your browser to view the interactive learning page and quiz.
- Edit the file directly to customize topics, questions, or styles, then reload the browser to see changes.
